# Impostorcitos-shared

Repositorio para poder almacenar la informacion de tipos, eventos, dtos, entre otros entre frontend y backend.

## Instalacion en el front y back

Para agregar el repo desde los otros repos de frontend y backend:
```bash
git submodule add https://github.com/SantiagoIvan/impostorcitos-shared.git shared
```

Cuando agregues nuevos objetos aca, en los otros repos tenes que ejecutar:
```bash
git submodule update --remote --merge
```
